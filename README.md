# IBM JavaScript  Full Stack Developer Capstone Project

## Project Overview

The capstone project creates a full-stack web application called "GiftLink". The GiftLink application connects users who wish to give away household items they no longer need with users who enjoy recycling and prefer to find free household items to match their tastes rather than purchasing new ones.

From the user interface perspective, the application contains a home page, a listings page, a navigation bar, a search function, an item details page, a registration page, a login page, and an editable profile page.


Project Tasks:

- Clone a GitHub directory containing the project's directory structure and skeleton code in respective files.
-  Develop user stories and acceptance criteria for the project.
- Develop front-end React components.
- Write APIs and endpoints in the back end and integrate those APIs into respective components on the front end.
-  Use MongoDB, a NoSQL database for data storage.
- Ensure a secure authentication and login process.
- At the end of the project,  containerize and deploy the application.
- Additionally, throughout the project, some common Agile and DevOps practices are implemented as part of the full-stack developer experience.
